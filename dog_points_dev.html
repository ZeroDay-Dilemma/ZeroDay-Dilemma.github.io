Winners Dog & Winners Bitch
Count the number of dogs of the gender in question. Use this number and refer to the Points Schedule.
Best of Winners
Calculate the points for WD and WB, and award BoW the higher of the two. If WD and WB both earn 0 points, BoW will get 1 point, should the sum of both classes be enough for 1 point based on the Points Schedule.
WD/WB is Best of Winners (WB/WD is BoS)
If BoS would recieve more points than BoW, BoW would also be awarded that number of points. Otherwise, no change.
WD/WB is Best of Opposite
Count the number of Class and Champions of the dogs sex. Use this number and refer to the Points Schedule.
WD/WB is Best of Opposite and BoW
If WD/WB would recieve more points than BoS, BoW would also be awarded that number of points. Otherwise, no change.
WD/WB is Best of Breed
Count the number of Class dogs of your sex, and all Champion dogs. Use this number and refer to the Points Schedule.
WD/WB is BoB and WB/WD is BoS
If BoS would recieve more points than BoB, BoB would also be awarded that number of points. This is because when a Class dog is BoB, it is automatically BoW. Otherwise, no change.
Champion is Best of Breed
Count the number of all Class dogs and all Champion dogs. Use this number and refer to the Points Schedule.
Champion is Best of Opposite
Count the number of Class and Champions of the dogs sex. Use this number and refer to the Points Schedule.
Select Dog/Bitch
Count the number of Class dogs of your sex, and all Champion dogs of your sex. Then, substract 1 from the resulting number. Use this number and refer to the Points Schedule.
